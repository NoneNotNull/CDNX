# CDNX
用于发现CDN背后的IP

```
usage: CDNX.py [-h] [--domain DOMAIN] [--network NETWORK] [--file FILE]
               [--keyword KEYWORD] [--browser]

CDN Bypass Explotion Tools

optional arguments:
  -h, --help         show this help message and exit
  --domain DOMAIN    Input the domain u want to find the true IP
  --network NETWORK  The network to scan,network[,network[,network]]
  --file FILE        The network file to scan
  --keyword KEYWORD  The keyword U wanna filter
  --browser          Scaned then browser it,default No
```
